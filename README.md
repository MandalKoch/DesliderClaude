## 🎯 What is this?

> No more *"so… what do you wanna play?"* for 45 minutes while the pizza gets cold.

**DesliderClaude** is a tiny web app your friend group uses to pick a boardgame together — **asynchronously**. Over the course of an afternoon, everyone opens a link on their phone, swipes 👍 / 👎 on the candidate games, and a live ranking updates as votes come in. When the host is ready, the top of the list is tonight's pick. 🎉

Not a review site. Not a rating service. Just: *"of these 12 games, which do we actually want to play tonight?"*

---

## 🚀 How it works

1. 🎟️ **Host creates a Game Night** — pick the candidate games, get a shareable link.
2. 👋 **Friends join from the link** — just a display name, no account.
3. 👉👈 **Everyone swipes at their own pace** — yes or no, change your mind any time.
4. 📊 **Live ranking** — anyone with the link sees the current leaderboard.
5. 🔒 **Host closes voting** — the top game wins. Play it tonight.

Install it to your home screen and it feels like a native app — no app store detour.
